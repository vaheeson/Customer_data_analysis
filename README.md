# Customer_data_analysis
Data Analytics Project with end-to-end industry standard Data Analysis of Customer Shopping Trends from Retail Data using SQL, Python and Power BI.

# 📊 Data Analytics Project

## 🔍 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to delivering actionable insights through dashboards and reports. The goal is to explore, clean, analyze, and visualize data using industry-standard tools and techniques.

---

## 📁 Dataset

* Source: Provided dataset (CSV/Excel format)
* Content: Customer transactions and behavioral data
* Features include:

  * Purchase details
  * Customer attributes
  * Ratings and discounts

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data loading, cleaning, and EDA
* **SQL** (PostgreSQL / MySQL / SQL Server) – Data querying and analysis
* **Power BI** – Interactive dashboard creation
* **Gamma** – Presentation (PPT) generation
* **PyCharm** – Development environment

---

## ⚙️ Project Workflow

### 1. Data Loading

* Imported dataset into Python using Pandas
* Verified structure, data types, and missing values

### 2. Exploratory Data Analysis (EDA)

* Summary statistics and distributions
* Identified trends, patterns, and anomalies
* Visualizations using charts and graphs

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types
* Standardized categorical values

### 4. SQL Analysis

* Loaded cleaned data into relational databases
* Executed analytical queries such as:

  * Aggregations (AVG, SUM, COUNT)
  * Conditional analysis using CASE statements
  * Percentage and trend calculations

### 5. Dashboard Development

* Connected database to Power BI
* Built interactive dashboards with:

  * KPIs
  * Filters and slicers
  * Visual insights (bar charts, pie charts, trends)

### 6. Reporting & Presentation

* Compiled findings into a structured report
* Created a presentation using Gamma to summarize insights

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone <your-repo-link>
cd <project-folder>
```

### 2. Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy pymysql
```

### 3. Run Python Scripts

* Open project in PyCharm
* Execute scripts for:

  * Data loading
  * Cleaning
  * EDA

### 4. Setup Database

* Create database in MySQL/PostgreSQL/SQL Server
* Import cleaned dataset
* Run SQL queries provided

### 5. Open Dashboard

* Launch Power BI
* Connect to your database
* Load the dashboard file (.pbix)
